## Returns & restocking-fee decision table

| Situation | Outcome |
|-----------|---------|
| Within **45 days** of delivery, resalable + original packaging | **Full refund, no fee** |
| Outside 45 days, within 26-month warranty, opened, normal return | **12% restocking fee** |
| Outside 45 days, within warranty, **confirmed hardware defect** | **Fee waived entirely** |
| Water / third-party-firmware / outdoor-mounting damage | **Not covered** — no refund |

Rule: the 12% restocking fee only applies to opened returns *outside* the 45-day
window but *within* the 26-month warranty — and it is waived when the reason is a
confirmed hardware defect.
